hexo+github pages博客搭建总结

发表于 2017-10-13 | 分类于 hexo

之前利用腾讯云搭建的个人博客，因为域名到期，不在使用，故重新使用github pages 和 hexo框架搭建了一个博客，这篇文章不会详细讲解基本的操作，会梳理一下整个流程以及教程分享。

1、环境的搭建

1. hexo汉语版官方文档: https://hexo.io/zh-cn/docs/index.html

  1. 文档中的注意点
    1. 安装好Node.js环境后，文档中的命令都在Node.js command promat中运行
    2. 为了防止_config.yml中的文件不被损坏，先备份此文件
    3. _config.yml中网站语言的基本参数language的值如果要设置为中文默认值为zh-Hans而不是zh-CH
    4. 每次配置好后，可以先使用命令hexo server启动本地的服务器，然后浏览器访问http://localhost:4000进行实时的预览
    5. 在文档的基本操作部分，Front-matter是指source/_post/xxx.md文件中的最上方的---区域，这里可以定义文档的标签tags和分类categories,标签下的目录(分类)是同级结构，而分类下的目录是非同级结构,具有层次性

2、 NexT主题的使用

1. NexT主题汉语版官方文档： http://theme-next.iissnan.com/getting-started.html

3、 Sublime Text及Markdown的使用

1. Markdown的基本语法： https://wizardforcel.gitbooks.io/markdown-simple-world/2.html

4、 博客的优化与定制

1. hexo搭建博客的深度优化与定制： https://github.com/heytxz/blog/issues/20

2. NEXT加载自定义的静态文件: http://www.joryhe.com/2016-05-22-create_the_custom_status_file.html
